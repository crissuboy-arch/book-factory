# Task: Posicionamento de Marca

## Objetivo
Definir onde a marca ocupa (ou deve ocupar) na mente do cliente — e criar uma estratégia para chegar lá e defender essa posição.

## Framework de Posicionamento

### Mapa de Posicionamento
Identificar os 2 eixos mais importantes para o avatar na escolha de produto nesta categoria:
```
Eixo X: [ex: Preço Baixo ←→ Preço Alto]
Eixo Y: [ex: Resultado Rápido ←→ Resultado Profundo]

Plotar:
- Concorrente 1: [X, Y]
- Concorrente 2: [X, Y]
- Concorrente 3: [X, Y]
- NOSSA MARCA: [X, Y — onde queremos estar]
- Espaço vazio identificado: [descrever]
```

### Declaração de Posicionamento (Neumeier)
```
Para [avatar específico]
que [necessidade ou desejo]
[nome da marca] é a [categoria ou frame de referência]
que [benefício diferenciador]
diferente de [principal alternativa]
porque [razão para acreditar].
```

### Estratégia de Categoria (Neumeier — ZAG)
**Opção 1: Competir na categoria existente**
- Risco: precisa ser significativamente melhor
- Estratégia: "O melhor [categoria] para [avatar específico]"

**Opção 2: Criar nova sub-categoria**
- Risco: educação de mercado necessária
- Estratégia: "O único [nova categoria] que [diferencial radical]"

**Opção 3: Redefinir a categoria**
- Risco: mais difícil, mais lento
- Recompensa: liderança de categoria garantida
- Estratégia: "Não somos [categoria antiga]. Somos [nova categoria que inventamos]."

## Output Esperado
- Mapa de posicionamento visual
- Declaração de posicionamento oficial
- Estratégia de categoria escolhida com justificativa
- 3 mensagens de posicionamento para diferentes canais
- Plano de 90 dias para estabelecer o posicionamento no mercado
