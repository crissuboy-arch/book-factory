# Task: Estrategia de Precificacao

## Objetivo
Definir o preço ideal para um produto ou serviço usando princípios de valor percebido, ancoragem e posicionamento de mercado.

## Principios de Precificacao (Hormozi)

### Regra #1: Preço = Posicionamento
- Preço alto → produto premium → cliente de maior qualidade
- Preço baixo → commodity → cliente que só quer desconto
- "Você não pode ser o mais barato E o melhor. Escolha."

### Regra #2: Calcule o ROI do Cliente
```
Valor que o cliente gera com o resultado: R$[X]
Tempo para o cliente gerar esse valor: [Y meses]
ROI esperado: [Z vezes o investimento]

Preço máximo aceitável: R$[X] ÷ 10 (cliente quer 10:1 ROI no mínimo)
Preço ideal: R$[X] ÷ 5 (5:1 ROI cria promotores)
```

### Regra #3: Ancoragem de Preço
Sempre apresentar antes do preço real:
1. O custo de NÃO resolver o problema
2. O custo de resolver com concorrentes/alternativas
3. O valor total do value stack
4. O preço especial de hoje

## Frameworks de Precificacao

### Modelo 1: Baseado em Valor
```
Dream Outcome do cliente: R$[valor em 12 meses]
Divisão por ROI esperado (10x): R$[preço máximo]
Desconto de risco percebido (50%): R$[preço realista]
Ajuste de mercado: R$[preço final]
```

### Modelo 2: Baseado em Concorrência + Premium
```
Referência de mercado (concorrente direto): R$[X]
Seu diferencial justifica premium de: [X]%
Seu preço: R$[X × (1 + premium)]
```

### Modelo 3: Ladder de Preços (3 opções)
Sempre oferecer 3 versões:
- **Básico** (R$[X]): produto/serviço mínimo sem bônus
- **Principal** (R$[X × 2-3]): oferta completa — o que 70% dos clientes escolhem
- **Premium** (R$[X × 5-10]): acesso direto, suporte estendido, implementação

*O objetivo do Básico é fazer o Principal parecer razoável.*
*O objetivo do Premium é fazer o Principal parecer acessível.*

## Output Esperado
- Cálculo de ROI do cliente com o produto
- 3 opções de preço com justificativa
- Ladder de preços (Básico/Principal/Premium)
- Script de apresentação de preço (como falar o número sem hesitar)
- Resposta para as 5 objeções de preço mais comuns
