# Task: Funil de Vendas Completo

## Objetivo
Projetar um funil de vendas de alta conversão que transforma visitantes frios em compradores recorrentes.

## Estrutura do Funil (Hormozi + Brunson)

```
TOPO: TRÁFEGO FRIO
│
├── Anúncio / Conteúdo Orgânico
│       Objetivo: 1 clique
│
↓
ISCA DIGITAL (Lead Magnet)
│   Entrega valor imediato + captura email
│   Taxa de conversão esperada: 30-50%
│
↓
SEQUÊNCIA DE BOAS-VINDAS (3-5 emails)
│   Aquece, educa, constrói autoridade
│   Duração: 5-7 dias
│
↓
OFERTA TRIPWIRE (Produto de Entrada)
│   Preço: R$27–R$97
│   Objetivo: Transformar lead em comprador
│   Converte desconhecidos em clientes
│
↓
CORE OFFER (Produto Principal)
│   Preço: R$297–R$2.997+
│   O produto central da empresa
│
↓
ORDER BUMP (na página de checkout)
│   Complemento que aumenta ticket médio 30-50%
│   Preço: 20-30% do core offer
│
↓
UPSELL 1 (imediatamente após a compra)
│   "Espere! Antes de ir — uma oferta única..."
│   Preço: 2-3x o core offer
│
↓
UPSELL 2 / DOWNSELL
│   Se recusou o Upsell 1, oferecer versão menor
│   Se aceitou, oferecer complemento premium
│
↓
BACKEND (clientes existentes)
│   Mentoria, mastermind, licenciamento
│   Preço: 10-50x o core offer
│
↓
REFERRAL SYSTEM
    Clientes satisfeitos viram fonte de leads gratuitos
```

## Métricas de Referência por Etapa

| Etapa | Métrica | Benchmark |
|-------|---------|-----------|
| Anúncio → Landing Page | CTR | 2–5% |
| Landing Page → Lead | Conversão | 30–50% |
| Lead → Tripwire | Conversão | 5–15% |
| Tripwire → Core Offer | Conversão | 20–40% |
| Core Offer → Order Bump | Conversão | 25–45% |
| Core Offer → Upsell 1 | Conversão | 15–35% |

## Output Esperado
- Mapa visual do funil completo
- Descrição de cada oferta em cada etapa
- Copy de cada página (headline + CTA mínimo)
- Métricas alvo para cada etapa
- Calculadora de LTV: quanto vale cada lead captado
