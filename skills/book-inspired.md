# Book Inspired Skill

## Description
A creative skill that generates original, new books inspired by the style, structure, and tone of reference works. This skill analyzes a reference book to extract key elements of writing style and narrative structure, then generates a completely original story with new characters, plot, and settings. All generated content is original and ethically suitable for commercial use.

## Supported Languages
- Portuguese (pt)
- English (en)
- Spanish (es)
- French (fr)

## What This Skill Does
- **Style Analysis**: Examines the reference book's writing style, vocabulary, sentence structure, and narrative voice
- **Structural Analysis**: Studies chapter length, pacing, narrative arcs, and plot progression
- **Tone Recognition**: Identifies emotional tone, mood, and thematic elements
- **Original Creation**: Generates completely new story with original characters, plot, settings, and ideas
- **Preservation of Essence**: Maintains the spirit and appeal of the reference work while avoiding any copying
- **Commercial Viability**: All content is original, non-infringing, and suitable for publication

## What This Skill Does NOT Do
- Copy plot, characters, or significant story elements from reference work
- Reuse dialogue, scenes, or specific narrative moments
- Create fan fiction or derivative works
- Violate intellectual property rights
- Plagiarize any content

## Supported Genres for Reference
- Romance and Contemporary Fiction
- Mystery and Thriller
- Fantasy and Science Fiction
- Historical Fiction
- Literary Fiction
- Adventure
- Paranormal and Supernatural

## Commands

### /book-inspired
Analyzes a reference book and creates an original story inspired by its style and structure.

#### Parameters
- `reference_book`: The title and author of the reference work (e.g., "The Notebook by Nicholas Sparks")
- `language`: The language for the new book (pt, en, es, fr)
- `genre`: The genre for the new book (can match or differ from reference)
- `chapters`: Number of chapters for the generated book (10, 20, 30, 60)
- `inspiration_aspects`: Which aspects to emphasize (e.g., "writing_style,pacing,emotional_depth")
- `original_elements`: User-specified elements (character types, settings, themes) to include

#### Output Structure
The inspired manuscript will include:
1. **Style Analysis Report**: Summary of reference book's writing characteristics
2. **Original Manuscript**: Complete new book with specified parameters
3. **Inspiration Documentation**: Clear tracking of what was analyzed vs. what was created
4. **Originality Certificate**: Confirmation that content is original and non-derivative
5. **Commercial Viability Assessment**: Evaluation for publication rights

#### Usage Example
/book-inspired reference_book="The Notebook by Nicholas Sparks" language=pt genre=romance chapters=20 inspiration_aspects="writing_style,pacing,emotional_depth" original_elements="protagonists:young professionals,setting:modern cities,theme:second chances"

## Analysis Framework

### Writing Style Elements
- **Vocabulary Level**: Simple vs. sophisticated, specialized terms
- **Sentence Structure**: Long vs. short sentences, complexity patterns
- **Narrative Voice**: First-person, third-person, omniscient, unreliable narrator
- **Dialogue Style**: Formal, casual, regional dialects, character-specific speech
- **Prose Quality**: Descriptive vs. sparse, metaphorical vs. literal
- **Unique Techniques**: Repeated devices, stylistic patterns, signature techniques

### Structural Analysis
- **Chapter Length**: Average word count per chapter
- **Pacing**: Fast-paced vs. slow-burn, action vs. introspection
- **Plot Progression**: Three-act structure, multiple subplots, climax intensity
- **Character Development**: Arc length, transformation pace, relationships
- **Time Progression**: Compressed vs. extended timeline, flashback usage

### Tone and Themes
- **Emotional Tone**: Optimistic, dark, humorous, dramatic, introspective
- **Primary Themes**: Love, redemption, justice, identity, power, family
- **Conflict Types**: Internal, external, romantic, societal, philosophical
- **Resolution Style**: Happy ending, bittersweet, open-ended, tragic

## Original Creation Process

### Stage 1: Analysis
- Read and analyze reference work
- Extract style, structure, and tone patterns
- Document key narrative techniques
- Identify universal appealing elements

### Stage 2: Original Framework
- Create new setting completely different from reference
- Develop original characters with unique backgrounds
- Design original plot with different core conflict
- Establish new themes aligned with user preferences

### Stage 3: Generation with Style Applied
- Write manuscript using analyzed style elements
- Apply identified pacing and structure patterns
- Emulate tone and narrative voice (without copying)
- Maintain original story integrity throughout

### Stage 4: Verification
- Confirm no copying of scenes, dialogue, or specific plot points
- Verify all characters, names, and setting are original
- Ensure thematic independence while maintaining inspiration
- Check for commercial viability and legal safety

## Inspiration vs. Copying: Clear Boundaries

### This IS Inspiration (Allowed)
- Similar emotional tone or mood
- Comparable chapter structure or length
- Matching narrative voice style
- Similar pacing and story progression
- Parallel thematic exploration
- Equivalent character arc complexity

### This Is NOT Allowed (Not Copied)
- Specific plot events or sequences
- Character names, descriptions, relationships
- Dialogue or narrative moments
- Specific settings, locations, landmarks
- Unique fictional world elements
- Particular scene compositions

## Legal and Ethical Framework
- **Non-Infringing**: Content doesn't violate copyright or intellectual property
- **Original Work**: New characters, plots, settings, and dialogue
- **Commercially Safe**: Suitable for self-publishing and distribution
- **Transparent**: Clear documentation of inspiration vs. creation
- **Author Intent Honored**: Captures essence without copying

## Limitations
- Cannot provide word-for-word recreations of reference material
- Analysis accuracy depends on book availability and quality
- User must provide or reference current books for analysis
- Some very distinctive story structures may limit originality
- Cultural or historical elements unique to reference work cannot be duplicated
